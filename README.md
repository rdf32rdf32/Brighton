# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a balanced 200-question one-at-a-time quiz with automatic progression, a seven-zone Brighton v Palace shoot-out with automatic Palace shots, player-controlled Bart Verbruggen saves, one-in-three Panenkas, full-screen play and statistics, compact month-filtered fixtures with calendar downloads, local kick-off time, matchday information, prediction and sharing tools, a four-formation XI builder with seven substitutes, Albion Story and Palace rivalry sections, Records & Honours, detailed Amex travel guidance, optional night-match theme, user-controlled anthem audio and offline support.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
