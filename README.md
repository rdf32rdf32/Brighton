# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a category-based 200-question quiz where the first choice is final, explanations remain visible before automatic progression and a visible answer timer shows what happens next. The seven-zone Brighton v Palace shoot-out includes automatic Palace shots, a freezing timing marker, clear timing verdicts, player-controlled Bart Verbruggen saves, realistic save techniques and slow-motion replays, a risky Panenka, saved shootout records, goalkeeper reports and post-match shot maps.

The site also includes compact mobile and landscape game layouts, full-screen play, clearly differentiated home and away fixtures, next-home and next-away cards, calendar downloads, local kick-off time, enhanced matchday information, match and league-position predictions, a four-formation XI builder with seven substitutes and a team-completion option, filtered Albion Story eras and Palace rivalry sections, Records & Honours, an interactive Amex stand finder and checklist, detailed travel guidance, master volume and reliable sound controls, sound captions, an optional night-match theme, progress memory, update notifications and offline support.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
