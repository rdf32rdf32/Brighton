# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a category-based 200-question quiz with one-at-a-time automatic progression and a visible answer timer, a seven-zone Brighton v Palace shoot-out with automatic Palace shots, a freezing timing marker, clear timing verdicts, player-controlled Bart Verbruggen saves, a risky Panenka, saved shootout records and post-match shot maps, compact mobile and landscape game layouts, full-screen play and statistics, spotlighted month-filtered fixtures with calendar downloads, local kick-off time, enhanced matchday information, match and league-position predictions, a four-formation XI builder with seven substitutes and a team-completion option, filtered Albion Story eras and Palace rivalry sections, Records & Honours, detailed Amex travel guidance, consistent save confirmations, optional night-match theme, user-controlled anthem audio, progress memory, update notifications and offline support.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
