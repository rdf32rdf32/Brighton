# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a 200-question one-at-a-time quiz, an animated Brighton v Palace alternating penalty shoot-out in which the player shoots and saves, compact month-filtered fixtures, matchday information, prediction tools, a four-formation XI builder with seven substitutes, Albion Story, Records & Honours, detailed Amex travel guidance, user-controlled anthem audio and offline support.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
