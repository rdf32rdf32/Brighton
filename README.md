# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a category-based 200-question quiz where the first choice is final, explanations remain visible before automatic progression and a visible answer timer shows what happens next. There is no visible Check answer step. The seven-zone Brighton v Palace shoot-out includes automatic Palace shots, a freezing timing marker, clear timing verdicts, player-controlled Bart Verbruggen saves, realistic save techniques and slow-motion replays, a risky Panenka, keyboard and game-controller support, saved shootout records, goalkeeper reports and post-match shot maps.

The v16 site adds a dynamic matchday and returning-supporter dashboard, a season database and progress centre, grouped search, a verified next-opponent briefing, prediction history, an expanded XI builder with captain, set pieces, tactics, drag swapping and recreatable links, an interactive Albion history search, a personal Amex itinerary and print view, information freshness labels, accessibility settings, local data export and restore, automatic recovery diagnostics, an opt-in guided tour and a maintenance dashboard for producing an updated central content file.

The site also includes compact mobile and landscape game layouts, full-screen play, clearly differentiated home and away fixtures, next-home and next-away cards, calendar downloads, local kick-off time, enhanced matchday information, match and league-position predictions, filtered Albion Story eras and Palace rivalry sections, Records & Honours, detailed travel guidance, master volume and reliable sound controls, sound captions, an optional night-match theme, update notifications and strengthened offline support.

Run `node site-check.js` before publishing. The included GitHub Actions workflow runs the same release checks automatically after uploads.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
