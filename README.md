# Albion Fan Hub

Unofficial static Brighton & Hove Albion supporter site for GitHub Pages.

Features include a category-based 200-question quiz with one-at-a-time automatic progression, a seven-zone Brighton v Palace shoot-out with automatic Palace shots, player-controlled Bart Verbruggen saves, a risky Panenka, a post-match shot map, compact mobile and landscape game layouts, full-screen play and statistics, compact month-filtered fixtures with calendar downloads, local kick-off time, matchday information, match and league-position predictions, a four-formation XI builder with seven substitutes, Today in Albion History, Albion Story and Palace rivalry sections, Records & Honours, detailed Amex travel guidance, optional night-match theme, user-controlled anthem audio, progress memory, update notifications and offline support.

Upload every file in the release ZIP directly to the GitHub Pages repository root.
